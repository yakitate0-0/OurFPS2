ゲーム名 : NIGHT BREAKER

Windows / Mac 

*************************************
操作方法
・WASDで移動する。
・シフトでしゃがみを行う。
・スペースでジャンプ。
・Rでリロード（銃の弾丸が切れたら自動リロードがされる）。
・マウスの右クリックでライト。
・左クリックで発砲。
*************************************

*************************************
キャラクターの説明
1. nomal BREAKER     平均的にパラメータ向上
2. Speed BREAKER     スピード向上
3. Jump BREAKER      ジャンプ力向上
4. Bullet BREAKER    弾数　30発
5. HP BREAKER        30秒に1回「x」で回復
6. Kill BREAKER      ２発でダウン可能
7. NIGHT BREAKER     暗視
8. Anti BREAKER      ある条件によってダメージ向上
*************************************
